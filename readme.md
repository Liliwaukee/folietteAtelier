## <a name="descripcion"></a> Descripción de producto:
- Foliette Atelier es un sitio responsivo que se compone de 6 secciones:
  - Productos
  - Pedidos especializados
  - Catálogo de telas
  - Tienda
  - Historia
  - Contacto
